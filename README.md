# Sistema de registro de ponto em Python

Esse é um projeto simples que eu fiz em Python para praticar programação.

A ideia do programa é cadastrar funcionários e registrar os horários de ponto do dia.

## O que o sistema faz

- Cadastra funcionários
- Registra 4 batidas de ponto por dia
- Guarda os horários em listas e dicionários
- Mostra os pontos registrados de cada funcionário

## Tecnologia usada

- Python

## Como rodar o programa

1. Baixar o projeto
2. Abrir o arquivo main.py
3. Executar o programa

## Demonstração
As capituras de telas mostram os processos feitos pelo sistema em ordem.

Cadastro de funcionario, registro de ponto e resutado final.
